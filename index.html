Projects

Tourney manager

Tourney Manager was a full stack collaborative project that was created during my Junior year of University. I collaborated with three other individuals and we used the Laravel framework and the SCRUM agile system to develop our website application. We met in-person every other week to discuss about any different algorithms or features that we wanted to implement. This website application was intended for users to be able to create an account and create a tournament of different teams that allowed other users to join. I was in charge of the graphic design, the system sequence diagrams, and some of the front-end development for this project.

Secret Santa

Secret Santa was a personal CRUD website that I created to facilitate a secret santa event for my family. This website allowed users to input, edit, or delete their information such as their sizes, interests, and favorite colors. All of the data was displayed in a table that contained every entry from every participant in each row. There was also an edit and delete button in each row so that the participants could change their interests/sizes if they wanted to. I built this website using SQL, PHP, CSS, and HTML. I also used inifinityfree to host the website and MySQL to manage my database.
